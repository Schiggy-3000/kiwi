2. Welche Daten werden verarbeitet?

Die App verarbeitet ausschliesslich Daten, die Sie selbst eingeben oder durch die Kamera aufnehmen:

  • Fotos von Belegen: Aufgenommen über die Kamera oder aus der Fotogalerie ausgewählt. Die Fotos werden ausschliesslich zur automatischen Texterkennung verwendet.
  • Belegdaten: Datum, Geschäftsname, Betrag, Mehrwertsteuer und Zahlungsart – automatisch erkannt oder manuell erfasst.

3. Zweck der Datenverarbeitung

Die erfassten Daten dienen ausschliesslich folgenden Zwecken:

  • Automatische Erkennung von Belegdaten mittels KI (Google Gemini API)
  • Lokale Speicherung der Belegdaten auf Ihrem Gerät
  • Export der gespeicherten Belege als CSV-Datei für eigene Buchführungszwecke

4. Weitergabe an Dritte

Beim Scannen eines Belegs wird das aufgenommene Foto zur Texterkennung an die Google Gemini API übermittelt. Dabei gelten die Datenschutzbestimmungen von Google (policies.google.com/privacy). Das Foto wird von Google nicht dauerhaft gespeichert.

Es erfolgt keine Weitergabe Ihrer Daten an andere Dritte, keine Werbung und kein Tracking.

5. Lokale Speicherung

Alle gespeicherten Belegdaten verbleiben ausschliesslich auf Ihrem Gerät (lokale SQLite-Datenbank). Es findet keine Cloud-Synchronisation statt. Bei der Deinstallation der App werden alle gespeicherten Daten gelöscht.

6. Übertragungssicherheit

Die Übertragung von Daten an die Google Gemini API erfolgt ausschliesslich über eine verschlüsselte HTTPS-Verbindung (TLS). Ihre Daten werden während der Übertragung geschützt.

7. Kamera- und Fotogalerie-Zugriff

Die App benötigt Zugriff auf:

  • Kamera: Um Fotos von Belegen aufzunehmen.
  • Fotogalerie: Um bestehende Fotos von Belegen auswählen zu können.

Diese Berechtigungen werden nur aktiv genutzt, wenn Sie die Scan-Funktion verwenden.

8. Keine Nutzerkonten, kein Tracking

Die App erfordert keine Registrierung und kein Benutzerkonto. Es werden keine Analyse- oder Tracking-Daten erhoben. Es werden keine Werbeanzeigen geschaltet.

9. Ihre Rechte

Da alle Daten lokal auf Ihrem Gerät gespeichert sind, haben Sie jederzeit die vollständige Kontrolle. Sie können gespeicherte Belege direkt in der App löschen oder durch die Deinstallation der App alle Daten entfernen.
